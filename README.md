# Py.PortScanner 🛡️

Bu proje, Python ve `socket` kütüphanesi kullanılarak geliştirilmiş temel bir ağ tarama (Port Scanner) aracıdır. Okul projesi kapsamında, ağ üzerindeki cihazların açık portlarını tespit etmek ve cihazlar arası veri iletişimini gözlemlemek amacıyla oluşturulmuştur.

## 🚀 Özellikler
- **IP & Domain Tarama:** Hem IP adresleri (örn. 192.168.1.x) hem de alan adları (örn. google.com) üzerinden tarama yapabilir.
- **Port Aralığı Belirleme:** Başlangıç ve bitiş portlarını kullanıcı tanımlı olarak tarar.
- **Hızlı Yanıt:** Timeout süresi ayarlanarak ağ gecikmelerine göre optimize edilebilir.
